# PV Simulator Challenge

## Requirements
- Docker + Docker Compose
- Python 3.12

## Steps to Run

### 1. Clone the repository
```bash
git clone https://github.com/krishnx/pv_simulator.git
cd pv_simulator
```

### 2. Build the Docker image
```bash
make build
make up
```

### 3. Run the tests
```bash
make test
```

### 4. Stop the Docker containers
```bash
make down
```

### 5. Check logs
```bash
make logs
```

### 6. View the output
```bash
make output
```
